# 下拉多选 formatSelect

主要用于在使用 el-select 组件的时候，当用户需要有 “全选” 选项的情况下，使用当前方法可以做到全部选中或者全不选中。方法源代码可以 点击这里。

# 使用方法

  • 需要添加一个变量 previewData 记录历史选项，下拉框 el-select 添加 @change 事件










 








  <el-select
    v-model="selectData"
    multiple
    collapse-tags
    filterable
    remote
    reserve-keyword
    placeholder="请输入关键词"
    :remote-method="remoteMethod"
    :loading="loading"
    @change="handleChange">
    <el-option label="全部" value="全部" v-if="totalData.length"></el-option>
    <el-option
     v-for="(item, key) in totalData"
     :key="key"
     :label="item.name"
     :value="item.id"></el-option>
  </el-select>

  • 在 handleChange 方法中调用公共方法中的 formatSelect，同时赋值给 previewData 历史记录。
  handleChange: function (value) {
    this.selectData = this.formatSelect(
      value,                  // 当前选中的数据
      this.totalData,         // 下拉框的总体数据
      this.previewData, 'id'  // 上一条历史数据
    )
    // 赋值给 previewData 历史记录
    this.previewData = this.selectData 
  },

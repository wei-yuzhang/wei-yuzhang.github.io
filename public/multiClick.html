# 防暴力点击

主要用于防止用户多次重复点击按钮，多次请求后台接口，影响系统性能。

# 使用方法

  • src/main.js 中添加以下代码
  // 防止暴力重复点击指令
  Vue.directive('preventReClick', {
    inserted (el, binding) {
      el.addEventListener('click', () => {
        if (!el.disabled) {
          el.disabled = true
          setTimeout(() => {
            el.disabled = false
          }, binding.value || 2000)
        }
      })
    }
  })

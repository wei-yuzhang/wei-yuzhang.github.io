# 移动适配

HTML5 引入了 name=“viewport” 属性来进行响应式布局，在介绍如何进行移动适配之前，请先了解一下 PPI、DPI、DPR 等概念。

# viewport

# 定义

设备的屏幕上用来显示网页的区域

通用设备上浏览器的默认 viewport 的宽度



# CSS 中的 1px 不等于设备的 1px

先看不同移动设备的像素、倍率：

  • 在桌面浏览器中二者是相等的
  • 在手机端根据 ppi 的不同，倍图也是不相同
  • 在用户对显示部分进行缩放

# 三个 viewport 理论

# 1、Layout viewport -- 浏览器窗口默认的 viewport

网页布局的区域，它是<\html>元素的父容器，通过 document.documentElement.clientWidth 来获取。

  • Layout viewport 用 CSS 像素来衡量尺寸，在缩放、调整浏览器窗口的时候不会改变。
  • 在桌面浏览器中，缩放为100%的时候，Layout viewport 宽度等于内容窗口的宽度。

在移动端，缩放为100%的情况下，Layout Viewport不一定等于内容窗口的大小。当你用手机浏览浏览宽大的网页（这些网页没有采用响应 式设计）的时候，你只能一次浏览网页的一个部分

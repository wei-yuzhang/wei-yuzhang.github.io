# git操作和使用

Git是一个开源的分布式版本控制系统，可以有效、高速地处理从很小到非常大的项目版本管理。

# Git简介与使用

  • 工作区
    电脑中可以看到的目录文件夹。
  • 版本库
    工作区有一个隐藏目录.git，这个不算工作区，而是Git的版本库。
  • 远程仓库
    Github上创建一个仓库，让这两个仓库进行远程同步。

# 提交过程

  • 使用git add把文件添加进去，实际上就是把文件修改添加到暂存区。
  • 使用git commit提交更改，实际上就是把暂存区的所有内容提交到当前分支。
  • 把本地库的内容推送到远程，用git push命令，实际上是把当前分支推送到远程仓库。

# 创建和合并分支原理

# 主要操作命令

  • 从远程库克隆项目：git clone 项目地址
  • 将工作区的内容放入版本库的暂存区：git add 内容
  • 将暂存区的内容提交到当前分支：git commit -m "git tracks changes"
  • 拉取分支：git pull
  • 推送分支：git push origin 对应的远程分支
  • 创建分支：git branch 分支名称
  • 切换分支：git checkout 切换到的分支
  • 创建分支，然后切换到分支：git checkout -b dev
  • 合并指定分支到当前分支：git merge dev(合并dev分支到当前分支)
  • 删除本地分支：git branch -d dev
  • 删除远程dev分支：git push origin :dev
  • 将当前版本回退到上一个版本: git reset --hard HEAD^
  • 回退到上两个版本：git reset – hard HEAD^^
  • 回退到指定版本：git reset --hard commit_id
  • 删除一个文件：git rm test.txt

    # 介绍

    一款基于vue.js的实用工具, 期望提升应用的开发效率，包括公共方法，vue 改进，js 封装。

    # 特性

    • 公共方法让我们在开发过程中可以重复使用其中的方法。
    • 使用Vue开发项目的优化以及问题解决方案。
    • 在使用JavaScript的时候，实现对JavaScript的方法封装。

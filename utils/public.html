# 下拉多选--formatSelect

formatSelect 方法主要用于在使用 el-select 组件的时候，当用户需要有 “全选” 选项的情况下，使用当前方法可以做到全部选中或者全不选中。

# 效果图



# 参数

  • key：el-option中 value 获取的值
  • value：当前选中的数据
  • remoteData：下拉框的总体数据
  • previewData：上一条历史数据

# 使用方法

  • 需要添加一个变量 previewData 记录历史选项，下拉框 el-select 添加 @change 事件










 








  <el-select
    v-model="selectData"
    multiple
    collapse-tags
    filterable
    remote
    reserve-keyword
    placeholder="请输入关键词"
    :remote-method="remoteMethod"
    :loading="loading"
    @change="handleChange">
    <el-option label="全部" value="全部" v-if="totalData.length"></el-option>
    <el-option
     v-for="(item, key) in totalData"
     :key="key"
     :label="item.name"
     :value="item.id"></el-option>
  </el-select>

  handleChange: function (value) {
    this.selectData = this.formatSelect(
      'id',                   // el-option中 value 获取的属性名
      value,                  // 当前选中的数据
      this.totalData,         // 下拉框的总体数据
      this.previewData        // 上一条历史数据
    )
    // 赋值给 previewData 历史记录
    this.previewData = this.selectData 
  },

# 格式化日期--formatDate

formatDate 方法主要用于在调用接口传参或者显示的时候，格式化成自己想要的日期格式。

# 参数

  • date：日期 -- Tue Aug 06 2019 00:00:00 GMT+0800 (中国标准时间)
  • fmt：返回的日期格式
    • y：年份
    • M：月份
    • d：日期
    • H：小时
    • m：分钟
    • s：秒数
    • q：季度
    • S：毫秒

# 使用方法

  this.formatDate(new Date(), 'yyyy-MM-dd HH:mm:ss')

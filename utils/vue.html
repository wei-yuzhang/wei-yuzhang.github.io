# 防暴力点击

主要用于防止用户多次重复点击按钮，多次请求后台接口，影响系统性能。

  • src/main.js 中添加以下代码

# 解决方案

  • 在 main.js 中添加防暴力点击指令
  // 防止暴力重复点击指令
  Vue.directive('preventReClick', {
    inserted (el, binding) {
      el.addEventListener('click', () => {
        if (!el.disabled) {
          el.disabled = true
          setTimeout(() => {
            el.disabled = false
          }, binding.value || 2000)
        }
      })
    }
  })

# IE 浏览器运行问题

# 问题1：提示 strict 模式下不允许一个属性有多个定义


这个问题根源在于重复定义某一个属性，就当前这个报错截图，可以看到文件中重复定义了 border 属性。

解决方案：找到对应的 vue 界面，删除重复 border 属性。

# 问题2：提示 Vuex requires a Promise polyfill in this browser

这个问题根源在于使用了ES6 Promise,而IE浏览器不支持

解决方案：使用 babel-polyfill

step1 安装 babel-polyfill：

  npm install --save-dev babel-polyfill
1

step2 在 webpack.config.js 中加入：

  module.exports = {
    entry: {app: ['babel-polyfill', './src/main.js']},

# js 更新日志

对 JavaScript 源码进行封装以便于以后的重复性使用，源代码请 点击这里。

# 1.0.1 (2019-08-06)

新增 Ajax 原生封装（兼容 IE 浏览器） 方法 -- wyz

  • 主要用于创建快速动态网页的技术。

新增 显示隐藏方法 -- wyz

  • 主要用于元素的显示隐藏。

新增 滑动方法 -- wyz

  • 主要用于元素的上下滑动。

新增 捕获/设置属性方法 -- wyz

  • 主要用于获取元素内容以及 html 标签

新增 设置样式方法 -- wyz

  • 主要用于可以设置单个也可以设置多个，如果存在例如 text-align 属性，需要写成 textAlign。

新增 添加元素方法 -- wyz

  • 主要用于在被选元素内部（外部）添加元素。

新增 删除/替换元素方法 -- wyz

  • 主要用于清空和替换元素。

新增 选择器方法 -- wyz

  • 主要用于根据 id 或者 class 获取元素。

新增 DOM 遍历方法 -- wyz

  • 主要用于获取元素的同胞，后代，祖先。

新增 防止重复点击方法 -- wyz

  • 主要用于防止重复点击。

新增 防抖和节流方法 -- wyz

  • 主要用于多次触发事件后，事件处理函数只执行一次，并且是在触发操作结束时执行。
  • 触发函数事件后，短时间间隔内无法连续调用，只有上一次函数执行后，过了规定的时间间隔，才能进行下一次的函数调用。

新增 鼠标事件方法 -- wyz

  • 主要用于获取鼠标的移动，滚动事件。

# public 更新日志

一款基于vue.js的公共方法, 期望提升应用的开发效率，源代码请 点击这里。

  • 共有两个文件，一个是 汉字转拼音，五笔，另一个是 common.js。
  • “汉字转拼音，五笔”文件夹中包括 汉字转拼音 和 汉字转五笔，可以查看对应源码。
  • common.js 文件在更新的时候注意 修改版本号（例如：版本号：1.0.1）。

# common.js (v1.0.1 2019-07-23)

新增 下拉多选 方法 -- wyz

  • 主要用于在使用 el-select 组件的时候，当用户需要有 “全选” 选项的情况下，使用当前方法可以做到全部选中或者全不选中。

新增 日期格式化 方法 -- wyz

  • 主要用于对获取的日期转换成自己需要的日期格式。

# 汉字转拼音五笔 (2019-08-13)

新增 汉字转拼音 方法 -- wyz

主要用于对用户输入的汉字自定义转换成拼音首字母以及拼音全拼。

新增 汉字转五笔 方法 -- wyz

方法主要用于对用户输入的汉字自定义转换成五笔。

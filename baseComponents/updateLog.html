# 更新日志

组件更新日志由提交记录生成，主要是组件的 bug 修复以及新增的组件特性简介。

# 1.0.1 (2019-07-05)

新增 接口日志（portLog） 组件 -- wyz

  • 主要用于在各个系统中查看接口发送以及接受结果，将日志中的错误信息展示在前台，更容易定位错误根源。

  • 日志组件功能：查询、高级搜索、表格分页、对话框

# 1.0.2 (2019-07-15)

新增 公用表格（tableCommon） 组件 -- sjy

  • 公用表格主要功能有选择和排序展示表格列、分页、展示从表、表格头配置筛选、表格头配置排序等功能。

# 1.0.3 (2019-07-26)

新增 图片预览（imagePreview） 组件 -- wyz

  • 图片预览组件主要对于图片进行查看、旋转、缩放、拖动等功能。

# 1.0.4 (2019-08-07)

接口日志组件新增 stateDisabled、dataInfoName 参数，同时对 iconData 参数 格式进行重新编辑 -- wyz

  • stateDisabled：返回接口的“状态”字段名。例如 state、status、success
  • dataInfoName：报文返回的字段名，总有四个参数
    • requestParameter：请求参数字段名
    • returnParameter：返回参数字段名
    • errorDetail：报错详情字段名
    • errorStatement：报错语句字段名
  • iconData：操作列字体图标，依次是报文、数据异常、重新发送
    • 报文按钮
      • flag：是否显示
      • icon：字体图标
    • 数据异常按钮
      • flag：是否显示
      • icon：字体图标
    • 重新发送按钮
      • flag：是否显示
      • icon：字体图标

# Attributes

参数 说明 类型 默认值 是否必须
isShow 图片预览组件是否显示 false：不显示 true：显示 Boolean false 是
count 显示哪一张图片(显示第一张，传入数字1)，默认显示第一张 Number 1 否
imageData 图片路径集合，通过数组格式传入 Array — 是

# Events

事件名称 说明 回调参数 是否必须
imageClose 图片预览关闭时触发(控制 isShow 属性值为 false即可关闭当前组件) — 是
